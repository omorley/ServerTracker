I am a student for course (comp2613) at BCIT, my processor is Henry Chan

The project I have created is a Server tracker, with relationships to power and cabinets in a Data Centre environment.

You can add/remove servers, cabinets and power circuits as needed

This application is powered by Swing backed by a H2 DB.

If you wish to see the data persisted to H2, open up the local console after the application has started. URL: http://localhost:8082 (H2 will start a server for you) and login with the following settings: Generic H2 (Embedded) org.h2.Driver jdbc:h2:mem:testdb blank username and password

Click ServerTracker.zip to download my application. After you unzip this application into a suitable folder, "cd" into that directory and run: java -jar ServerTracker-final.jar

The code for this project can be found via github https://github.com/omorley/ServerTracker
